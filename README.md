# XQEbot 功能菜单

上次更新: 2026-02-20 17:41:20

## 📋 功能概览

XQEbot 是一个功能丰富的QQ机器人，提供多种实用功能，包括但不限于主功能、插件功能、小说查询、娱乐功能等。本菜单提供了所有可用命令的详细说明。

## 🔍 快速搜索

使用 `Ctrl+F` 在本页面搜索您需要的功能。

## ⭐ 主功能

### 基础命令

- `/helps` - 生成帮助图, 别名：帮助、菜单、功能
- `/全部功能` - 查询全部功能
- `/bot菜单` - 打开bot菜单
- `/bot状态` - 查询bot状态
- `/status` - 获取bot状态（高级版，且生成时间较长）
- `/密码转md5 <明文密码>` - 密码转MD5

### 排行榜功能

- `/今日排行榜` - 查看今日发言排行榜【其他关于排行榜的功能请输入：/描述-排行榜】

### 其他功能

- `/sofree` - 查看一只sofree
- `/BOT赞我` - 让bot赞你
- `/fuck` - 命令纠错插件，灵感来源于著名的命令行工具 thefuck。当你不小心输错了一个命令时（例如 /pixic 而不是 /pixiv），只需紧接着发送 /fuck，此插件就会尝试找出你可能想输入的正确命令，并给出建议。

### 语录管理

## 👑 群管功能

### 成员管理

- `/禁言 <秒数> @用户` - 对指定成员禁言
- `/禁我 <秒数>` - 对自己禁言
- `/解禁 @用户` - 解除指定成员的禁言
- `/开启全禁 / 关闭全禁` - 控制全群是否可发言
- `/改名 <新昵称> @用户 / 改我 <新昵称>` - 修改群名片
- `/头衔 <头衔> @用户 / 申请头衔 <头衔>` - 设置群头衔（需群主权限）
- `/踢了 @用户` - 将指定成员移出群聊
- `/拉黑 @用户` - 踢出并拉黑指定成员
- `/上管 @用户` - 设置管理员（需群主权限）
- `/下管 @用户` - 取消管理员（需群主权限）

### 消息管理

- `/撤回 (引用消息) / 撤回 @用户 数量` - 撤回消息，默认10条
- `/设精 (引用消息) / 移精 (引用消息)` - 管理精华消息
- `/查看精华` - 查看精华消息列表

### 群设置

- `/设置群头像 (引用图片)` - 修改群头像
- `/设置群名 <新群名>` - 修改群名称
- `/发布群公告 <内容> (可引用图片)` - 发布群公告
- `/查看群公告` - 查看群公告

### 投票功能

- `/投票禁言 <秒数> @用户` - 发起禁言投票
- `/赞同禁言 / 反对禁言` - 投票同意或反对禁言

### 安全设置

- `/设置禁词 <词1 词2…>` - 设置或查看自定义违禁词
- `/内置禁词 开/关` - 开启或关闭内置违禁词检测
- `/刷屏禁言 <秒数>` - 设置刷屏触发的禁言时长（0 关闭）
- `/开启宵禁 HH:MM HH:MM` - 设置宵禁时间段
- `/关闭宵禁` - 关闭宵禁任务

### 进群管理

- `/进群审核 开/关` - 开启或关闭进群审核功能
- `/进群白词 <关键词…>` - 设置自动批准进群的关键词
- `/进群黑词 <关键词…>` - 设置自动拒绝进群的关键词
- `/未命中驳回 开/关` - 设置是否拒绝未命中白词的进群申请
- `/进群等级 <等级>` - 设置进群等级门槛
- `/进群次数 <次数>` - 设置最大进群尝试次数
- `/进群黑名单 +QQ号 / -QQ号` - 添加或移除进群黑名单
- `/进群禁言 <秒数>` - 设置新进成员的禁言时长（0 表示不启用）
- `/进群欢迎` - 开启/关闭进群欢迎消息
- `/退群通知 开/关` - 开启或关闭退群通知
- `/退群拉黑 开/关` - 开启或关闭退群后自动拉黑（拉黑后下次进群自动拒绝）
- `/批准 / 驳回 <理由>` - 审批进群申请

### 群数据管理

- `/群友信息` - 查看群成员活跃情况
- `/清理群友 <未发言天数> <群等级>` - 移除不活跃或低等级成员
- `/上传群文件 <文件夹名/文件名>` - 引用文件并上传
- `/删除群文件 <文件夹名或序号> <文件名或序号>` - 删除群文件或文件夹
- `/查看群文件 <文件夹名或序号> <文件名或序号>` - 查看文件夹或文件详情
- `/取名 @用户 <抽取消息轮数>` - 根据聊天记录取个群昵称
- `/群管重置 <群号/all>` - 重置本群或全部群的群管配置
